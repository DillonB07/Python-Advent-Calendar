Welcome to the eighteenth door! It's now Saturday 18th December, and so it is time for your eighteenth Python challenge in this advent calendar. Your task is below:

All of these presents and necessities that need buying are expensive! Your job is to make a finance calculator that gets given a user-inputted budget. Then add up the entered finances and tell the user whether they're in debt or they have enough money.
Your program should do the following:

  • Accurately calculate finances.

  • Take a user-inputted budget

However, you could add the following:

  • Tell the user how much budget they have left.

  • Let the user take off items.

  • Save the data to a file and open it when the program restarts.

  • Be unique! Instead of just using an unstyled Terminal window, make your program as a website, app or style the Terminal using the curses module. Or, just add colour, animation and clearing the screen!
