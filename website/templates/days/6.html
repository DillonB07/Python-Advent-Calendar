Welcome to the sixth door! It's now Monday 6th December, and so it is time for your sixth Python challenge in this advent calendar. Your task is below:

Santa wants to know what the weather will be like in a given location to prepare for his trip around the world on Christmas Eve. He would like you to make a program that uses a weather API and the requests module to find the weather at any location in the world. Here is a good API service, but you could use any others - OpenWeatherMap API
Your program should do the following:

  • Use the requests module to get data from an API.

  • Take user input for the city to get the weather for.

  • Present the data from the API nicely instead of just printing out the JSON response.

However, you could add the following:

  • Use other features from the API.

  • Ask the user if they want to store the daya in a file

  • Be unique! Instead of just using an unstyled Terminal window, make your program as a website, app or style the Terminal using the curses module. Or, just add colour, animation and clearing the screen!
