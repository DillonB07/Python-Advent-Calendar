Welcome to the second door! It's now Thursday 2nd December, and so it is time for your second Python challenge in this advent calendar. Your task is below:

Santa and his elves want to communicate with each other without humans understanding. Create a program that will translate and decode messages into Morse Code.
Your program should do the following:

  • Allow the user to enter a string which will be encoded into Morse code using '.' and '-' notation. Spaces should be replaced by the piper character('|').

  • The program should then translate inputted Morse code back into English.

  • The program should have a menu to choose between encoding and decoding.

However, you could add the following:

  • Change the program to be more Christmas themed.

  • Create your own language to translate into.

  • Be unique! Instead of just using an unstyled Terminal window, make your program as a website, app or style the Terminal using the curses module. Or, just add colour, animation and clearing the screen!
