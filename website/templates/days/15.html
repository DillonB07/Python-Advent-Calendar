Welcome to the fifteenth door! It's now Wednesday 15th December, and so it is time for your fifteenth Python challenge in this advent calendar. Your task is below:

It's time for a new game! Or rather, a way of making games. Your task is to make a program that lets the user input questions and then save it to a file. The program should also have a menu where you can choose from the following: view quizzes, play quizzes or create quizzes.
Your program should do the following:

  • Allow Santa and the elves to create quizzes and save them to a separate file.

  • Allow saved quizzes to be easily loaded.

However, you could add the following:

  • Add an option to modify quizzes

  • Be unique! Instead of just using an unstyled Terminal window, make your program as a website, app or style the Terminal using the curses module. Or, just add colour, animation and clearing the screen!
