Welcome to the seventh door! It's now Tuesday 7th December, and so it is time for your seventh Python challenge in this advent calendar. Your task is below:

GIFs. Kids like them. Teens like them. Adults like them. Everyone likes them. Who knows how to make them? Santa is considering giving out flash drives full of GIFs for presents, but he doesn't know how to make them. Your task is to create an easy-to-use tool that converts small video files into GIFs. Have a look at this to get started! - A Python guide to converting videos to GIFs.
This project is a little different, because it is quite tricky to do in a basic project. You could use an API to do this and return the download link to the user. Or, you could use another language like JavaScript which might be easier for this project.
Your program should do the following:

  • Convert popular video formats(.mov, .mp4) into .gifs

  • Be simple to use

  • Allow the user to download the file if possible.

However, you could add the following:

  • Allow the user to crop the video

  • Allow the user to freeze a part of the video

  • Make your GIF time symmetrical

  • Allow the user to add text to the video

  • Allow the user to remove the background of the video

  • Be unique! Instead of just using an unstyled Terminal window, make your program as a website, app or style the Terminal using the curses module. Or, just add colour, animation and clearing the screen!
