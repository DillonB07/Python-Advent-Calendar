Welcome to the ninth door! It's now Thursday 9th December, and so it is time for your ninth Python challenge in this advent calendar. Your task is below:

Money. It's possible to make £2.50 with 1x£1, 2x50p, 1x5p and 3x1p. Make a program that takes a given amount and lists all the possible coin combinations to make that amount.

Your program should do the following:

  • Allow the user to enter an amount of money and accurately list ALL of the combinations using real coins(1p, 2p, 5p, 10p, 20p, 50p, £1, £2)

However, you could add the following:

  • Use notes as well as coins (£5, £10, £20, £50, £100, £200, £300, £500, £1000)

  • Be unique! Instead of just using an unstyled Terminal window, make your program as a website, app or style the Terminal using the curses module. Or, just add colour, animation and clearing the screen!
