Santa has a lot of people to deliver presents to. Yesterday, you gave him a way of knowing whether they deserve a present! Create a program to work out what presents they should get. This could integrate into yesterday's solution. It should take in a name at a time and whether they were naughty or nice. This should then be put into a list or dictionary like so: 


naughty: int = 0
nice: int = 1
result = {
    'Henry': naughty,
    'Dillon': nice
} # Don't hardcode the list! This is just an example of how it should be stored!

Your program should do the following:

  • Tell Santa what presents the kids deserve

  • Store the results in a list or dictionary/JSON object.

However, you could add the following:

  • Allow the user(Santa) to view and modify the list or dictionary/JSON object

  • Allow bulk deciding of users.
